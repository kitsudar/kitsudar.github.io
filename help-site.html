SCRAM Systems Help Site

The Problem

A static help documentation page containing individual PDF files causing security and usability concerns. There's no structure to the help page with the files that are impossible to search and find.

The Solution

Provide a content management system for the training and documentation team so they can easily manage and publish the content in Wordpress.

The Project

Implemented Wordpress and build out a documentation management site for the company's training and documentation team. Site hierarchy and framework were built to support all the documentation sections.
