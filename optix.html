Optix Platform

The Problem

As the company continues to expand the software product offerings, there is not a way for customers to manage all the products on one platform.

The Solution

Provide a unified platform for customers to be able to manage all their products from one place. The platform allows for a single entry point as well as surface the information across the products that the customer is managing.

The Project

Design the visual and interactive components of Optix while the user can personalize the dashboard. Optix surfaces information from different underlying applications with a modern interface. Optix provides a way for users to manage, access, and navigate into the current system applications.
